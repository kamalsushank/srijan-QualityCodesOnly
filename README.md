## 📌 Overview

The **AI Testing Dashboard** is an intelligent web-based platform that simulates a real-world testing workflow similar to tools like **Jira + Selenium**.

It allows users to:
- Select a project  
- Explore its features (user stories)  
- Automatically generate test cases using AI  
- Simulate execution of those test cases  
- Analyze risks and confidence levels  

---

## 🎯 Problem We Are Solving

In traditional software development:
- Writing test cases is time-consuming  
- Test coverage may be incomplete  
- Risk assessment is often manual and inconsistent  
- Teams lack quick insights into feature reliability  

### ❗ Our Goal:
To automate and simplify the testing lifecycle using AI, making it:
- Faster  
- Smarter  
- More insightful  

---

## ⚙️ What This Project Does

This dashboard provides an end-to-end AI-driven testing flow:

### 🔹 Step 1: Select Project
Choose from predefined projects like:
- Online Shopping App  
- Mental Health Chatbot  
- Notes Sharing App  

### 🔹 Step 2: Select Feature (User Story)
Each project contains features like:
- Login  
- Add to Cart  
- Payment  
- Chatbot interaction  
- Notes sharing  

### 🔹 Step 3: AI Test Generation
AI automatically generates:
- Positive test cases  
- Negative test cases  
- Edge cases  
- Risk level (Low / Medium / High)  
- Initial confidence score  

### 🔹 Step 4: Test Execution (Simulated)
Executes test cases (simulated or Selenium-ready)

Shows:
- Pass/Fail results  
- Execution details per test  

### 🔹 Step 5: Final Analysis
- Final confidence score (based on execution)  
- Priority level (Critical / Moderate / Low)  
- Visual insights for decision-making

## 🧠 How It Works

### 🖥️ Frontend (React)
- Displays projects and features  
- Handles user interaction  
- Sends requests to backend  

Renders:
- Test cases  
- Execution results  
- Confidence graphs  

### 🔙 Backend (Django / DRF)
- Provides project + feature data  
- Uses AI API (OpenRouter / LLM)  
- Generates structured test cases  
- Simulates execution logic  

Calculates:
- Risk  
- Confidence scores  
- Priority levels
